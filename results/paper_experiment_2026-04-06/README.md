# Paper Experiment Results

This directory stores the structured outputs generated for the thesis experiment run on `2026-04-06`.

Included files:

- `cnnp_psnr_results.csv`: complete per-image result table
- `per_image_results.csv`: duplicate per-image table kept for asset grouping
- `average_results.csv`: average metrics grouped by mode and payload
- `environment_info.txt`: runtime environment and seed information
- `payload_psnr_trend.csv`: source data for the payload-vs-PSNR chart
- `mode_gap_comparison.csv`: source data for the mode gap chart
- `payload_psnr_trend.png`: generated trend chart
- `mode_gap_comparison.png`: generated mode gap chart
- `logs/*.txt`: raw run snapshots for each `mode + payload` combination
